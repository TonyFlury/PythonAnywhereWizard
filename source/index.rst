Welcome to the documentation for the pyAnyWhereWizard.

The Wizard is a Desk top application to allow structured and repeated control of
your Python Anywhere installations. The Wizard is currently a simple Proof of concept but eventually it will :

* Synchronise with the PythonAnyWhere server using the API
* Allow deployment of arbitrary commands to the servers
* Scripting template engine to allow repetition of commands on Demand
* Audit trail for the commands and scripts
* Ability to add custom scripts
* Configuarable UI to add scripts to the UI
* Integrtion with GITHub to identify branches/release for deployment
